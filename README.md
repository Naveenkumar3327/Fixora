## ✨ Overview

Fixora is a modern service marketplace platform that bridges the gap between customers and local service professionals.

Whether it's an electrician, plumber, mechanic, carpenter, AC technician, cleaner, or home maintenance expert, users can instantly discover nearby professionals, compare ratings, book services, track appointments, and communicate directly within the app.

Built with a premium user experience, advanced animations, real-time location services, and scalable architecture, Fixora delivers a marketplace experience inspired by modern products such as Uber, Airbnb, and Urban Company.

## 🚀 Getting Started

### Clone Repository

```bash
git clone https://github.com/yourusername/fixora.git
```

### Navigate to Project

```bash
cd fixora
```

### Install Dependencies

```bash
flutter pub get
```

### Run Application

```bash
flutter run
```

---

## 📦 Environment Setup

Create a `.env` file:

```env
GOOGLE_MAPS_API_KEY=YOUR_API_KEY
FIREBASE_PROJECT_ID=YOUR_PROJECT_ID
```
